# Code for tree mortality and soil carbon study

This repository contains the R scripts used to regenerate the numerical results and figures reported in the study on topographically decoupled tree mortality disturbances, microbial strategies and divergent soil organic carbon (SOC) fates in a subtropical forest.

## Contents

- `R/`: R scripts for data processing, statistical analyses and figure generation  
- `data/`: curated input tables required by the scripts  
- `README.md`: brief documentation  
- other folders: described in the comments of the R scripts

## How to use

1. Download or clone this repository.  
2. Open R (or RStudio), set the working directory to the repository root.  
3. Run the scripts in the order indicated in the comments to reproduce the main and supplementary results.
